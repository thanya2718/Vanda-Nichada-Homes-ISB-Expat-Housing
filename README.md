# Vanda Nichada Homes ISB Expat Housing

Next.js bilingual website for GitHub Pages.

## Local test

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build test

```bash
npm run build
```

## Deploy

Push to the `main` branch. GitHub Actions will build and deploy to GitHub Pages.

Important: this project is configured for the repository name:

`Vanda-Nichada-Homes-ISB-Expat-Housing`

If you change the repository name, update `repoName` in `next.config.js`.
